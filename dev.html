Homebanking - MindHub Brothers

Proyecto final de Bootcamp de Java.

  • Backend desarrollado con spring boot
  • Frontend desarrollado con vue
  • Base de datos postgreSQL

Cuenta con una base de datos donde se almacenan los usuarios, las tarjetas, cuentas y préstamos. Verifica los datos para que un usuario no pueda ver ni acceder a cuentas de otros. Todos los datos de entrada están validados desde el backend.

Para entrar crear un usuario nuevo o usar mirta@mindhub.com con clave tito Es un homebanking. Permite crear usuarios nuevos, crear hasta 3 cuentas y eliminarlas, pedir y borrar tarjetas y tomar préstamos.
